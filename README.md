Gesture Recognition for Sign Language Translation using Deep Learning and Computer Vision

Overview
This project implements a real-time gesture recognition system for American Sign Language (ASL) and Indian Sign Language (ISL). Using deep learning and computer vision, the system translates physical gestures into text and speech output, enabling accessible communication for the hearing-impaired community.

Features
Dual-model support: ASL and ISL (switchable in the web app).
Real-time gesture recognition using MediaPipe and OpenCV.
Output layer includes:
Text display of recognized gesture.
Speech synthesis via gTTS.
Web deployment using Flask with interactive templates.
Modular codebase for easy extension and reproducibility.
